Smart Bus

Vision

    Smart Bus is an AmI system that targets a specific category of public transport users to help them in their daily usage, it provides information to the users about the bus before it arrives e.g. when will the bus arrive at a given stop, if the bus is empty of crowded and the air quality in the bus, which allow the users to make a decision either waiting for the bus or searching for another commute solution to reach the destination.

    Smart Bus main feature is for the users who require help to get on/off the bus like people who use wheelchairs, the system offers them more options to improve their experience and make it more independent by interacting with the bus through the system.

    The wheelchair users can choose the bus they are going to take and by doing that the system will inform the driver about them so he can take necessary actions to help them if needed, the system also will alert the bus passengers to free the area reserved for wheelchairs using a recorded voice message and flashing lights that surround the area that needs to be free to facilitate getting on or off the bus.

    The system includes a mechanical slider that will be extended automatically to reach the sidewalk to allow the wheelchair user to get on the bus in a fast, easy and more comfortable way, so the driver will not need to leave his seat to help unless itâ€™s an emergency.

    The system also provides an emergency report system to inform and alert the driver to take the necessary actions.

AmI main steps

Sensing

The system can sense if the area reserved for wheelchairs is free or not, can also sense if the bus is crowded and the air quality inside the bus.

Reasoning

The system will use the data collected from the sensors, and based on it will provide the users with helpful information.

Acting

The system uses an alert system to alert other passengers to free the reserved area for wheelchairs, also a mechanical slider to facilitate the movement of wheelchairs from the sidewalk to the bus and vice versa.

Interacting

The users can interact with the system through a mobile app to get useful information and use different actions based on the user profile.

AmI features

Sensitive

Different sensors are used to collect different types of data, motion sensors in the reserved area for wheelchairs, Thermal sensors, air quality sensors.

Intelligent

It transforms the sensors data into useful information that is used to take the proper action to guide users to free the reserved area or clear a path for the wheelchair to get on/off the bus, also it gives suggestions to open the windows based on the bus's air quality.

Responsive

The system can deal with many users at the same time and provide and offer useful information based on the continuously collected data from the sensors.

Adaptive

The system adapts to different users and provide different services to them based on their profiles.

Transparent

The users can interact with the system actions while the system itself is invisible to them.

Ubiquitous

The users can interact with the system from anywhere before they can get on the bus.
